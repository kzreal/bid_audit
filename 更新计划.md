## 一、目标

将投标文件审核系统的数据存储从 localStorage 迁移到 SQLite 数据库，实现：

1. 项目数据持久化存储
2. 支持 Word 文件二进制存储
3. 切片内容、审核任务、审核结果完整保存
4. 选择项目后可加载历史数据进行审核

---

## 二、技术方案

**SQLite 操作方式**：使用 Python 内置的 `sqlite3` 模块（无需安装额外 MCP）

- Python 3 自带 `sqlite3` 模块
- 直接在 `backend_server.py` 中使用
- 简单高效，无额外依赖

**不使用 MCP 的原因**：
- Python sqlite3 是内置模块，零配置
- 直接集成到 Flask 后端，代码更简洁
- 避免 MCP 调用开销

- 类型：SQLite

### 2.2 表结构

```sql
-- 项目表
CREATE TABLE projects (
    id TEXT PRIMARY KEY,
    name TEXT NOT NULL,
    file_name TEXT,
    file_size INTEGER,
    slice_level INTEGER DEFAULT 0,
    slice_count INTEGER DEFAULT 0,
    created_at TEXT NOT NULL,
    updated_at TEXT NOT NULL,
    last_opened_at TEXT
);

-- Word 文件表（存储二进制文件）
CREATE TABLE word_files (
    id TEXT PRIMARY KEY,
    project_id TEXT NOT NULL,
    file_data BLOB NOT NULL,
    file_name TEXT NOT NULL,
    created_at TEXT NOT NULL,
    FOREIGN KEY (project_id) REFERENCES projects(id) ON DELETE CASCADE
);

-- 切片表
CREATE TABLE slices (
    id TEXT PRIMARY KEY,
    project_id TEXT NOT NULL,
    index_num INTEGER NOT NULL,
    title TEXT,
    level INTEGER DEFAULT 0,
    content TEXT,
    start_line INTEGER,
    end_line INTEGER,
    FOREIGN KEY (project_id) REFERENCES projects(id) ON DELETE CASCADE
);

-- 审核任务表
CREATE TABLE tasks (
    id TEXT PRIMARY KEY,
    project_id TEXT NOT NULL,
    title TEXT NOT NULL,
    description TEXT,
    status TEXT DEFAULT 'pending',
    sort_order INTEGER DEFAULT 0,
    created_at TEXT NOT NULL,
    updated_at TEXT NOT NULL,
    FOREIGN KEY (project_id) REFERENCES projects(id) ON DELETE CASCADE
);

-- 审核结果表
CREATE TABLE reviews (
    id TEXT PRIMARY KEY,
    task_id TEXT NOT NULL,
    project_id TEXT NOT NULL,
    conclusion TEXT,
    reason TEXT,
    evidence TEXT,
    bid_source TEXT,
    requirement_source TEXT,
    slices_reviews TEXT,
    created_at TEXT NOT NULL,
    FOREIGN KEY (task_id) REFERENCES tasks(id) ON DELETE CASCADE,
    FOREIGN KEY (project_id) REFERENCES projects(id) ON DELETE CASCADE
);
```

# 数据库连接
DB_PATH = 'data/bid_review.db'

def get_db():
    """获取数据库连接"""
    conn = sqlite3.connect(DB_PATH)
    conn.row_factory = sqlite3.Row
    return conn

# 项目 CRUD
